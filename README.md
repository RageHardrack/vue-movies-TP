# Vue Movies App

A [YouTube Tutorial by Tyler Potts ](https://www.youtube.com/watch?v=UHewcsv6uJY) using Vue JS, the [Open Movie Database API](http://www.omdbapi.com) and the Composition API.

## Dependencies:
- Vue JS : 3.0.0
- Vue-router : 4.0.0-0
- SASS : 1.26.5

## Project setup
```
npm install
```
